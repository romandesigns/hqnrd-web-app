A theme toggle component that allows users to switch between light, dark, and system themes.

## Returns

`Element`

A dropdown menu component with theme selection options (Light, Dark, System)

## Remarks

This component renders a dropdown menu with a button that displays a sun/moon icon
based on the current theme. The icons animate smoothly during theme transitions.
